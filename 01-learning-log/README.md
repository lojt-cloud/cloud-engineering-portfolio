# Cloud Engineering Learning Log

My daily notes, labs, and small exercises from the Ironhack Cloud Engineering bootcamp.
This is a working journal, not a polished portfolio piece — see my other pinned repos for
the cleaned-up project work.

## Phases
- `prework/` — Linux, networking, Git/GitHub basics
- `cloud-foundations/` — IAM, S3, pricing/cost basics
- `networking-architecture/` — VPC/VNET, subnets, load balancers
- `iac-automation/` — Terraform, GitHub Actions, Ansible
- `observability-finops/` — Prometheus, Grafana, cost analysis
- `security-compliance/` — Vault, compliance frameworks, security baselines

Each subfolder has its own README with what I built, what I learned, and what was hard —
these notes double as drafts for my LinkedIn posts.
