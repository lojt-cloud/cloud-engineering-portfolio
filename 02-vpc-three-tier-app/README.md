# Three-Tier App on a Single Cloud VPC

## What this is
A secure three-tier application architecture (web / app / data layers) deployed inside
a single cloud provider's VPC, built as part of Ironhack's Networking & Single-Cloud
Architecture module.

## Tools used
- AWS / Azure (VPC, subnets, NAT gateway, load balancer)
- Terraform (provisioning)
